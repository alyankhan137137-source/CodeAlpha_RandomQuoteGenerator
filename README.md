# 📖 Random Quote Generator

A beautiful, minimal, and clean Flutter application that inspires you with random quotes every time you open it or click a button. Developed as part of the CodeAlpha internship task.

---

## ✨ Features

- **Instant Inspiration:** Displays a random quote as soon as the app starts.
- **Refresh Capability:** Tap the "New Quote" button to instantly fetch another inspiring quote.
- **Elegant UI:** Features a clean, minimal design with a soft gradient background and high-quality typography.
- **Responsive Layout:** Works seamlessly across different screen sizes.
- **Well-Structured Code:** Built with a modular architecture for easy maintenance.

---

## 🛠️ Tech Stack

- **Framework:** [Flutter](https://flutter.dev/)
- **Language:** [Dart](https://dart.dev/)
- **UI Components:** Material 3 Design
- **State Management:** StatefulWidget (Standard Flutter)

---

## 📂 Project Structure

The project follows a modular structure to keep the code organized:

```text
lib/
├── data/
│   └── quote_data.dart     # Contains the list of quotes
├── models/
│   └── quote.dart          # Data model for a Quote object
├── screens/
│   └── quote_screen.dart   # Main UI and logic
├── widgets/
│   └── quote_display.dart  # Reusable UI component for quotes
└── main.dart               # Entry point of the app
```

## 🚀 Getting Started

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install) installed on your machine.
- An IDE (Android Studio, VS Code, or IntelliJ).

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/CodeAlpha_RandomQuoteGenerator.git
   ```

2. **Navigate to the project folder:**
   ```bash
   cd CodeAlpha_RandomQuoteGenerator
   ```

3. **Get dependencies:**
   ```bash
   flutter pub get
   ```

4. **Run the app:**
   ```bash
   flutter run
   ```
